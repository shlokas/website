CODATRON

A multilevel coding competition which includes:
  1. GAME OF CODES:-An online pre-event to Codatron conducted on HackerRank/Codechef.
  2. QUALIFIER-A two step qualifying process for the semi finals which consist of debugging and programming each of 15 and 30 minutes respectively.
  3. THE RELAY (Semi finals)-Teams of two participate in the relay. Each member is given a question to solve them simultaneously and both must finish to start solving the next!
  4. THE FINAL BATTLE-The finale consists of two sub events: blind coding-where the team must type the entire code with the monitors turned off-and the final code. The final code is the last step towards victory with the highest level of difficulty.
