VIRTUAL PLACEMENTS

Because of this same motive we have decided not to eliminate any participant in any of the rounds. This is one of the appealing aspects of the event which will attract participants. The rounds that we plan to conduct are:

Aptitude test (Hackerrank)
Technical Skills
