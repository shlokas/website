LINE FOLLOWING BOT

Line following bot is an autonomous robot which follows a black line on a white background.
  • Robot must be able to detect particular line and keep following it.
  • This Robot should sense the line with its sensors that are installed under the robot.
  • Winners will be decided based upon the time required to conquer the given track with minimum number of penalties made.
